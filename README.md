# West-Bengal-Election-2026-Analysis
# Election Results Analysis Report

## Project Title

**Election Results Analysis and Constituency Insights Dashboard**

## Objective

The objective of this project was to analyze election results data to understand voting patterns, candidate performance, party dominance, and constituency-level trends using data analytics and visualization techniques.

## Dataset Overview

The dataset contains detailed election information, including:

* Constituency Number and Name
* Candidate Name
* Political Party
* EVM Votes
* Postal Votes
* Total Votes Received
* Vote Share Percentage

The dataset consists of multiple candidates competing across different constituencies, providing valuable insights into electoral performance and voter behavior.

## Data Preprocessing

The following preprocessing steps were performed:

* Imported the dataset into Google Colab using Pandas.
* Checked for missing and duplicate values.
* Verified data types and column names.
* Cleaned and formatted the data for analysis.

## Key Performance Indicators (KPIs)

The project analyzed several important election metrics:

* Total number of votes cast.
* Total number of candidates.
* Total number of political parties.
* Total number of constituencies.
* Candidate with the highest votes received.
* Highest vote share percentage.

## Visualizations Performed

### 1. Top 10 Parties by Total Votes

A bar chart was used to identify political parties with the highest total votes.

### 2. Vote Share Distribution

A histogram was created to understand the distribution of vote share percentages among candidates.

### 3. Party-wise Vote Share Analysis

A box plot was used to compare vote share performance across political parties.

### 4. Top 10 Candidates by Total Votes

A horizontal bar chart highlighted candidates with the strongest voter support.

### 5. EVM Votes vs Postal Votes

A scatter plot was used to analyze the relationship between EVM votes and postal votes.

### 6. Winning Candidates Analysis

The project identified candidates receiving the highest number of votes and compared their performances.

### 7. Party Performance Analysis

The total votes secured by each party were analyzed to determine party dominance.

### 8. Constituency-Level Insights

The project examined voting trends and candidate performance across constituencies.

## Major Findings

* A few major political parties dominated the election in terms of total votes.
* Most candidates received relatively lower vote shares, while a small number of candidates achieved exceptionally high support.
* EVM votes contributed the majority of total votes, whereas postal votes represented a smaller proportion.
* Significant variation existed in candidate performance across constituencies.
* The highest vote-getting candidate in the dataset received strong voter support with a vote share exceeding 60%.

## Conclusion

This project successfully transformed raw election data into meaningful insights through data cleaning, analysis, and visualization techniques. The findings helped identify top-performing candidates, dominant political parties, and constituency-level voting patterns.

Overall, this project provides a comprehensive overview of election trends and serves as an effective example of real-world data analytics using Python and Google Colab.
